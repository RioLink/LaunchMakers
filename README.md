# NEXORA Digital

Багатосторінковий статичний сайт SEO-агенції.

## Запуск
Запустіть через VS Code Live Server або `python -m http.server 8000`.

## Форми
Фронтенд-валидація працює. Для реальної відправки підключіть WordPress/Contact Form 7, PHP handler або CRM webhook.

## Перед публікацією
Замініть демонстраційні контакти, домен, ціни, кейси та соціальні посилання.
